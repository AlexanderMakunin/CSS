Diferencias entre las clases (class) e identificadores (id) en CSS

Las clases se usan agrupar y modificar en conjunto mientras que los identificadores se usan para modificar ciertas cosas concretas

Con html se hace tipo <"p" class ="clase" id="identificador"><"/p"> y en el css para el identificador #identificador{} y clase .clase{}
